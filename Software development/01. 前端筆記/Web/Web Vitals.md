#### 核心指標
- TTFB (Time to first byte) 首次位元組時間
- FCP (First comtentful paint) 首次內容繪製
- LCP (Largest contentful paint) 最大內容繪製
- TTI (Time to interactive) 可互動時間
- CLS (cumulative layout shift) 累積佈局偏移
- FID (First input delay) 首次輸入延遲
